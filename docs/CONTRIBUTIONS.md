# Contributions Guide

Thank you for your interest in contributing to this project!

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Make your changes** following the existing code style and conventions.
3. **Test your changes locally:**
   - Run `dotnet build` from the solution root.
   - Start the app via `AspireApp.AppHost` to verify all services.
4. **Commit your changes** with a clear message.
5. **Open a Pull Request** describing your changes and validation steps.

## Guidelines

- Keep changes focused and well-scoped.
- Do not add secrets or credentials.
- Run all services via `AspireApp.AppHost` for validation.
- Manual testing is required before submitting a PR.

## Contributor Recognition

All contributors are automatically recognized on the GitHub repository page.
